==========
Installing
==========

Fire up Octave/Matlab and add the folder m in your path::

  addpath('path_to_shapetoolbox/m')

That's it.  Add that command to your startup-file, either
``.octaverc`` (for Octave) or ``startup.m`` (for Matlab).
